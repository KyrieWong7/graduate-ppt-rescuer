# Create Presentation

Two modes:

1. **Paper-derived slides**
   - derive narrative from the paper's source-of-truth
   - prioritize contribution, design, results, and takeaways
2. **Standalone slide project**
   - build directly under `projects/presentations/`
   - use Beamer or Quarto as the primary source

## Required Reviews

- visual audit
- pedagogy review
- Quarto QA if translated

## Deliverables

- slide source
- slide pdf
- html / Quarto output when requested
